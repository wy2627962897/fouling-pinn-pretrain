# 换热器结垢热阻基础

## 一、热阻串联结构

换热器传热：Q = ΔT / R_total

R_total 由 5 层热阻串联组成：

| 序号 | 热阻 | 类型 | 公式 | 运行中变化 |
|---|---|---|---|---|
| ① | 管内对流热阻 | 对流传热 | 1/(hi·Ai) | 基本不变 |
| ② | 管内污垢热阻 | 污垢导热 | Rf,i | **唯一持续增长** |
| ③ | 管壁导热热阻 | 热传导 | δw/(kw·Am) | 不变（制造确定） |
| ④ | 管外污垢热阻 | 污垢导热 | Rf,o | **唯一持续增长** |
| ⑤ | 管外对流热阻 | 对流传热 | 1/(ho·Ao) | 基本不变 |

**两种传热方式：**
- 热传导：Q = (kA/δ)·ΔT，热阻 = δ/(kA)
- 对流传热：Q = h·A·(Tw − T∞)，热阻 = 1/(hA)

## 二、Rf 增长的工程后果

```
Rf ↑ → R_total ↑ → 为维持换热量 Q 不变 → 必须增大 ΔT → 多烧燃料/多耗电
```

操作员无法增大 hi/ho（泵功代价大）或增加换热面积（已建成），只能提高温差。

## 三、清洗决策优化

**核心矛盾：**

| 策略 | 停产损失 | 能耗损失 |
|---|---|---|
| 频繁清洗 | 大 | 小 |
| 不频繁清洗 | 小 | 大 |

**最优清洗周期**：使 min{能耗损失(t) + 清洗费用(t)} 的时间点。

**关键前提**：必须知道 Rf(t) 的演化规律。不知道就相当于盲猜。

## 四、结垢动力学：Kern-Seaton 模型

```
dRf/dt = φd − φr
```

- φd（沉积速率）：盐类析出 → 附着壁面 → 逐层堆积。受温度（Arrhenius）、浓度（过饱和度）、物质传递（流速输送）控制
- φr（剥离速率）：流体剪切力将表面沉积物刮走。受壁面剪切应力 τs 和当前污垢厚度控制

**流速的双重角色：**
- 流速↑ → 输送更多沉积物到壁面 → φd↑
- 流速↑ → 剪切力增大 → φr↑
- 低流速区 φd 主导，高流速区 φr 主导

**Kern-Seaton 具体形式：**

```
dRf/dt = A·exp(−Ea/RTw) − B·τs·Rf
```

φd = 常数（给定工况），φr = K·Rf（正比于当前污垢量）

## 五、Rf(t) 的演化规律

因 φd 恒定，φr = K·Rf：

- t = 0 时，Rf = 0，φr = 0，净增长 = φd（**最大**）
- 随 Rf 增大，φr 增大，净增长逐渐减小
- 当 K·Rf = φd 时，净增长 = 0，达到**渐近值**

**解的形式：**

```
Rf(t) = Rf,∞ · (1 − e^(−t/τ))
```

其中：
- Rf,∞ = φd / K（渐近污垢热阻）
- τ = 1/K（时间常数）

**工况对参数的影响：**
- 壁温↑ → φd 指数级增大 → Rf,∞ 变大，τ 变短
- 流速↑ → K 增大 → Rf,∞ 变小，τ 变短
- 浓度↑ → φd 增大 → Rf,∞ 变大

## 六、纯物理模型的局限

Kern-Seaton 假设 φd 和 K 是常数，但工业现场：

- 进料温度波动 → φd 实时变化
- 流速调整 → K 实时变化
- 水质波动 → 结垢机制可能切换

**→ 纯物理模型预测不可靠。需要 AI（PINN）将物理约束与数据驱动结合。**
